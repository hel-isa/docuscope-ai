# DocuScope AI

Privacy-first hybrid AI document fingerprinting.

DocuScope AI scans folders and subfolders, extracts document metadata and content insights, redacts sensitive data, classifies document types, and generates sanitized JSON and Excel reports.

## MVP
- Scan folders recursively
- Support PDF, DOCX, XLSX, TXT, JPG, PNG
- Detect and mask sensitive data
- Hybrid classification: rules first, AI fallback
- Generate sanitized document fingerprints
- Export JSON + Excel
